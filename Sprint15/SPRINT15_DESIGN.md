# SPRINT15_DESIGN.md

# Boussole -- Les Gardiens des Repères

> **Version : V1 (Conception fonctionnelle)**
>
> Ce document définit la vision officielle du Sprint 15. Il décrit
> l'expérience utilisateur, les règles de conception et les décisions
> validées. Il ne contient volontairement aucune décision technique.

------------------------------------------------------------------------

# 0. Vision

## Pourquoi Boussole existe

Boussole aide les enfants à développer leur autonomie grâce à des
repères positifs.

Le parent prépare le cadre.

Le Gardien accompagne.

L'enfant reste toujours libre de ses choix.

La mission du Gardien est simple :

**Toujours aider l'enfant à trouver son prochain petit pas.**

------------------------------------------------------------------------

# Les principes fondateurs

-   Le Gardien ne remplace jamais les parents.
-   Il ne donne jamais d'ordre.
-   Il propose toujours.
-   Il accompagne.
-   Il valorise les efforts.
-   Il ne culpabilise jamais.
-   Il ne compare jamais deux enfants.
-   L'autonomie est plus importante que la performance.

------------------------------------------------------------------------

# Les Gardiens des Repères

Compagnons disponibles dès le premier lancement :

-   Crystal -- calme, rassurant.
-   Pixel -- créatif, inventeur.
-   Pyro -- courageux, protecteur.
-   Gear -- organisé, méthodique.
-   Wave -- doux, empathique.

L'enfant peut changer librement de Gardien.

Aucune perte de progression.

------------------------------------------------------------------------

# La Maison

La maison devient l'accueil de l'espace enfant.

Elle appartient à l'enfant.

Le Gardien y vit.

La maison évolue avec la progression de l'enfant.

Elle change selon :

-   l'heure réelle
-   le jour
-   les saisons (plus tard)
-   la décoration
-   l'activité du Gardien

------------------------------------------------------------------------

# Progression

La progression appartient à l'enfant.

Jamais au Gardien.

Les décorations, souvenirs et évolutions restent définitivement acquis.

------------------------------------------------------------------------

# Les Éclats

Les Éclats représentent les petits progrès réalisés.

Ils ne sont jamais offerts gratuitement.

Ils récompensent les étapes importantes.

Jamais l'ouverture de l'application.

Ils servent à personnaliser la maison et créer des moments de vie.

------------------------------------------------------------------------

# Les Moments Partagés

Ils remplacent les mini-jeux.

Exemples :

-   préparer un chocolat
-   observer les étoiles
-   construire une cabane
-   lire une histoire
-   faire un cadeau aux parents
-   aider à préparer un repas
-   décorer la maison

------------------------------------------------------------------------

# Les Missions Secrètes

Événements rares proposés principalement :

-   mercredi
-   week-end
-   vacances
-   fêtes
-   événements familiaux
-   création par le parent

Le Gardien les présente comme un secret.

L'enfant réalise la mission.

Le parent valide.

Les Éclats sont attribués uniquement après validation parentale.

------------------------------------------------------------------------

# Moteur des Repères

Le Gardien analyse :

-   heure
-   planning
-   routines
-   centres d'intérêt
-   temps disponible
-   besoins

Puis il propose exactement deux possibilités.

Jamais une.

Jamais plus de deux.

------------------------------------------------------------------------

# Les besoins

-   Trouver une idée.
-   Retrouver un repère.
-   Être rassuré.
-   Trouver du courage.
-   Retrouver son calme.
-   Célébrer un progrès.

------------------------------------------------------------------------

# Navigation enfant

Accueil = Maison.

Depuis la maison :

-   Ma journée
-   Les Trouvailles
-   Moments partagés
-   Changer de Gardien

------------------------------------------------------------------------

# Modes d'appareil

## Téléphone familial

Choix du profil.

## Tablette personnelle

Ouverture directe sur un enfant.

## Tablette partagée

Choix parmi les enfants autorisés.

------------------------------------------------------------------------

# PIN parent

Créé uniquement lors de l'activation du premier appareil en mode enfant.

Accès parent :

Maintenir le logo Boussole pendant 5 secondes.

Puis saisir le PIN.

Cette méthode est rappelée dans les paramètres parent.

------------------------------------------------------------------------

# Anti-triche

Une journée ne peut distribuer ses Éclats qu'une seule fois.

Réinitialiser une journée ne redonne jamais de récompense.

Les Missions Secrètes suivent la même règle.

------------------------------------------------------------------------

# Ce que Boussole ne sera jamais

-   un Tamagotchi
-   un jeu mobile à récompenses infinies
-   un assistant qui remplace les parents
-   une IA qui décide pour l'enfant
-   une application culpabilisante
-   une application addictive

------------------------------------------------------------------------

# Vision finale

L'objectif n'est pas que l'enfant ouvre Boussole pour gagner des Éclats.

L'objectif est qu'il dise :

> « Regarde, mon Gardien sait toujours quoi me proposer quand je ne sais
> plus quoi faire. »

Et que les parents puissent dire :

> « Grâce à Boussole, nous créons plus de beaux moments en famille. »
