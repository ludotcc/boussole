
# MAISON.md
# Documentation officielle – Sprint 15
Version : 2.0

> La Maison est le véritable écran d'accueil de l'enfant. Ce n'est pas un menu mais un lieu vivant où réside son Gardien.

---

# 1. Vision

Chaque enfant possède une seule maison.

La maison appartient à l'enfant.

Les Gardiens viennent y vivre, mais ils n'en sont jamais propriétaires.

La progression est liée à l'enfant.

---

# 2. Objectifs

La maison doit donner l'impression :

- d'être vivante ;
- d'évoluer avec le temps ;
- de raconter l'histoire de l'enfant ;
- d'être rassurante.

---

# 3. Principes

- Une seule maison par enfant.
- Changer de Gardien ne change jamais la maison.
- Les décorations restent.
- Les souvenirs restent.
- Les Éclats servent à l'embellir.
- Aucun bonus de gameplay n'est accordé par les décorations.

---

# 4. Accueil

Lorsque l'enfant ouvre Boussole :

Le Gardien est déjà occupé.

Il peut :

- lire ;
- cuisiner ;
- jardiner ;
- bricoler ;
- dessiner ;
- ranger ;
- observer les étoiles ;
- préparer une Mission Secrète ;
- se reposer.

Le monde existe même sans l'enfant.

---

# 5. Cycle de la journée

## Matin

- Lever du Gardien
- Ouverture des volets
- Ambiance lumineuse
- Préparation de la journée

## Journée

Le Gardien alterne plusieurs activités.

## Soir

Le Gardien ralentit.

La lumière devient plus chaude.

## Nuit

Si la journée est terminée :

- le Gardien dort ;
- les lumières sont tamisées ;
- aucune nouvelle activité n'est proposée.

---

# 6. Pièces

Version initiale

- Salon
- Coin lecture
- Cuisine
- Jardin

Évolutions futures

- Chambre
- Atelier
- Terrasse
- Potager
- Observatoire

---

# 7. Les Trouvailles

Les Trouvailles remplacent la boutique.

Catégories :

- Mobilier
- Plantes
- Décorations
- Luminaires
- Accessoires
- Objets rares
- Souvenirs

Chaque objet possède :

- un prix en Éclats ;
- une description ;
- une rareté.

---

# 8. Évolution

La maison évolue lentement.

Chaque évolution représente les progrès de l'enfant.

Exemples :

- nouvelle fenêtre ;
- nouveau tapis ;
- nouvelle plante ;
- nouveau meuble ;
- nouvelles décorations.

La progression est prévue pour plusieurs années.

---

# 9. Souvenirs

Chaque Gardien laisse un souvenir permanent.

Crystal :
Lampe étoilée

Pixel :
Petit robot

Pyro :
Lanterne

Gear :
Horloge

Wave :
Coquillage

Ces objets ne disparaissent jamais.

---

# 10. Interactions

Depuis la maison, l'enfant peut accéder à :

- Ma journée
- Les Trouvailles
- Moments partagés
- Changer de Gardien

Le Gardien guide naturellement ces accès.

---

# 11. Ambiance

La maison doit toujours transmettre :

- calme ;
- sécurité ;
- chaleur ;
- bienveillance.

Aucune interface agressive.

Peu de texte.

Le Gardien reste l'élément principal.

---

# 12. Règles de conception

La maison n'est pas un village.

La maison n'est pas un jeu de gestion.

La maison est le foyer du Gardien.

Chaque objet acheté doit raconter une histoire.

---

# 13. Vision finale

L'enfant doit parfois ouvrir Boussole uniquement pour dire bonjour à son Gardien.

S'il ne fait aucune activité ce jour-là, la visite doit malgré tout être agréable.

La maison doit devenir un lieu familier où l'enfant aime revenir naturellement.
