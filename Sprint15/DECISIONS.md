# DECISIONS.md

# Registre officiel des décisions -- Sprint 15

> Ce document centralise toutes les décisions validées pendant la
> conception du Sprint 15. Une décision inscrite ici est considérée
> comme validée et ne doit être remise en question que si une nouvelle
> décision officielle la remplace.

------------------------------------------------------------------------

# Vision

-   Boussole devient un compagnon numérique qui aide les enfants à
    développer leur autonomie.
-   Le compagnon accompagne l'enfant, il ne remplace jamais les parents.
-   L'objectif est d'aider l'enfant à toujours trouver son prochain
    petit pas.

------------------------------------------------------------------------

# Les Gardiens

-   Les 5 Gardiens sont disponibles dès le premier lancement.
-   Aucun Gardien n'est supérieur à un autre.
-   L'enfant peut changer librement de Gardien.
-   Le changement ne nécessite pas de validation parentale.
-   Le Gardien accompagne l'enfant, mais la progression appartient
    toujours à l'enfant.

Ordre officiel des Gardiens :

1.  Crystal
2.  Pixel
3.  Pyro
4.  Gear
5.  Wave

------------------------------------------------------------------------

# La Maison

-   La maison devient l'accueil officiel de l'espace enfant.
-   La maison appartient à l'enfant.
-   Les décorations restent même après un changement de Gardien.
-   Les souvenirs restent définitivement.
-   Le Gardien vit selon l'heure réelle.
-   Lorsqu'une journée est terminée, le Gardien dort.

------------------------------------------------------------------------

# Les Éclats

-   Les points deviennent les Éclats.
-   Les Éclats représentent les progrès réalisés.
-   Ils ne sont jamais distribués pour une simple connexion.
-   Ils récompensent les étapes importantes.
-   Une journée ne peut distribuer ses Éclats qu'une seule fois.
-   Les Éclats servent à personnaliser la maison et débloquer des
    moments de vie.

------------------------------------------------------------------------

# Missions Secrètes

-   Les Missions Secrètes sont des événements rares.
-   Elles apparaissent principalement les mercredis, week-ends, vacances
    ou événements particuliers.
-   Le parent peut également en créer.
-   L'enfant valide la mission.
-   Le parent confirme sa réalisation.
-   Les Éclats sont attribués uniquement après validation parentale.

------------------------------------------------------------------------

# Moteur des Repères

-   Le Gardien analyse le contexte avant de répondre.
-   Il ne donne jamais d'ordre.
-   Il propose toujours exactement deux possibilités.
-   L'enfant reste libre de choisir.
-   Si les deux propositions ne conviennent pas, le Gardien peut en
    proposer deux nouvelles.

------------------------------------------------------------------------

# Navigation

-   Deux expériences coexistent : Parent et Enfant.
-   L'espace enfant est totalement indépendant.
-   La maison est la page d'accueil de l'enfant.

------------------------------------------------------------------------

# Modes d'appareil

Trois modes officiels :

-   Téléphone familial
-   Tablette personnelle enfant
-   Tablette partagée

Le PIN est créé uniquement lors de l'activation du premier mode enfant.

Accès parent :

Maintenir le logo Boussole pendant 5 secondes puis saisir le PIN.

Cette procédure est rappelée dans les paramètres parent.

------------------------------------------------------------------------

# Règles fondamentales

-   Ne jamais culpabiliser un enfant.
-   Toujours valoriser les efforts.
-   Le compagnon accompagne, il ne commande jamais.
-   Les parents gardent toujours le dernier mot.
-   Les récompenses ne doivent jamais créer de dépendance.
-   Chaque nouvelle fonctionnalité doit renforcer l'autonomie et les
    liens familiaux.
