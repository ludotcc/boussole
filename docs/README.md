# Boussole

> **Des repères pour bien grandir.**

Boussole est une application Flutter destinée aux familles.

Son objectif est d'aider les enfants à développer leur autonomie grâce à un compagnon numérique bienveillant, tout en réduisant la charge mentale des parents.

---

# Vision

Le parent prépare le cadre.

Le Gardien accompagne.

L'enfant avance.

Boussole ne remplace jamais les parents.

Elle crée des repères, des habitudes positives et des souvenirs familiaux.

# Philosophie

Boussole est :

- bienveillante ;
- rassurante ;
- simple ;
- motivante.

Boussole ne sera jamais :

- un Tamagotchi ;
- un jeu mobile addictif ;
- une IA remplaçant les parents ;
- une application culpabilisante.

---

# État du projet

Le Sprint 15 constitue la nouvelle fondation du projet.
