# PROJECT_CONTEXT.md

# Projet

Nom : Boussole

Slogan :
Des repères pour bien grandir.

---

# Mission

Boussole accompagne les enfants dans leurs routines quotidiennes en proposant une expérience positive, rassurante et ludique.

Le parent prépare le cadre.

L'enfant vit l'expérience.

---

# Utilisateurs

- Parent
- Enfant

Les parents configurent.

Les enfants utilisent.

---

# Valeurs

- Bienveillance
- Autonomie
- Simplicité
- Encouragement
- Confiance
- Aucun jugement

---

# Identité graphique

Style :

- doux
- lumineux
- formes rondes
- illustrations aquarelle
- mascotte hibou
- couleurs pastel

Ne jamais casser cette identité.

---

# Mascotte

Le hibou est le guide.

Il accompagne.

Il encourage.

Il rassure.

Il ne donne jamais d'ordres.

---

# UX

Toujours :

- une action principale
- peu de texte
- progression visible
- grosses zones tactiles
- messages positifs

Jamais :

- surcharge
- culpabilisation
- comparaison entre enfants
