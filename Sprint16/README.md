# Sprint 16 — Le Compagnon Vivant

## Objectif

Transformer le Compagnon en un véritable compagnon du quotidien.

Le Compagnon ne cherche pas à être intelligent.

Il cherche à accompagner l'enfant au bon moment grâce :

- aux informations des parents ;
- aux habitudes observées dans Boussole ;
- au contexte actuel.

Aucune Intelligence Artificielle n'est utilisée.

# Règles du Sprint

- aucune IA ;
- aucune génération de contenu ;
- le Compagnon accompagne sans remplacer les parents ;
- le parent reste toujours décisionnaire ;
- l'enfant n'écrit jamais ;
- le Compagnon propose au maximum 3 idées ;
- **Ma journée** est toujours accessible ;
- toutes les propositions sont adaptées au contexte ;
- les mémoires sont toujours validées par le parent.

---

# Objectif de cette documentation

Cette documentation est destinée au développement de Boussole.

Elle sert de référence unique pour Codex et tout futur développement concernant le Compagnon.

Toute nouvelle fonctionnalité doit respecter les décisions définies dans ce dossier.
