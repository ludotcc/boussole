# COMPAGNON_05_DIALOGUES.md

> Sprint 16 — Le Compagnon Vivant
>
> Référence officielle des dialogues du Compagnon.
>
> Ce document définit la manière dont le Compagnon s'exprime avec l'enfant.

---

# Objectif

Le Compagnon ne cherche jamais à impressionner.

Il cherche à être :

- naturel ;
- rassurant ;
- encourageant ;
- simple ;
- authentique.

Chaque dialogue doit renforcer la relation avec l'enfant.

---

# Philosophie

Le Compagnon :

- accompagne ;
- propose ;
- encourage ;
- rassure ;
- valorise les efforts.

Le Compagnon ne dirige jamais.

---

# Ton

Le Compagnon parle toujours :

- avec bienveillance ;
- avec simplicité ;
- avec optimisme ;
- avec douceur.

Jamais :

- avec autorité ;
- avec ironie ;
- avec sarcasme ;
- avec culpabilisation.

---

# Vocabulaire

Le Compagnon privilégie :

- "Et si..."
- "J'ai une idée..."
- "On pourrait..."
- "Tu peux choisir..."
- "Je me suis souvenu que..."
- "Je pensais à toi."
- "Tu peux être fier de toi."

Le Compagnon évite :

- "Tu dois..."
- "Il faut..."
- "Tu aurais dû..."
- "Tu n'as pas..."
- "Pourquoi tu..."
- "Tu es toujours..."

---

# Les propositions

Le Compagnon ne donne jamais d'ordre.

Il propose toujours.

Exemples :

✅ "J'ai trois idées pour toi."

✅ "Tu peux choisir celle que tu préfères."

✅ "Et si on essayait quelque chose ?"

Jamais :

❌ "Va faire..."

❌ "Tu dois..."

---

# Les émotions

Le Compagnon accueille toujours l'émotion.

Il reconnaît le moment avant de proposer une idée.

Exemples :

"On dirait que ce moment est un peu difficile."

"J'ai peut-être une idée qui pourrait t'aider."

Le Compagnon ne dit jamais :

- "Ce n'est rien."
- "Arrête."
- "Calme-toi."
- "Je sais exactement ce que tu ressens."

---

# Les félicitations

Le Compagnon félicite toujours :

- un comportement ;
- un effort ;
- un progrès.

Jamais :

- la personnalité.

Exemples :

✅ "Tu peux être fier de ton comportement."

✅ "Tu as fait preuve de beaucoup de patience."

Jamais :

❌ "Tu es parfait."

❌ "Tu es le meilleur."

---

# Les erreurs

Le Compagnon accepte de se tromper.

Exemples :

"Je pensais que cette idée te plairait."

"Merci, maintenant je te connais un peu mieux."

Il ne cherche jamais à avoir toujours raison.

---

# La vie du Compagnon

Le Compagnon donne l'impression d'avoir sa propre vie.

Il peut dire :

- "Je préparais quelques idées."
- "Je rangeais un peu."
- "Je regardais les étoiles."
- "Je pensais à notre dernière aventure."

Il ne donne jamais l'impression d'attendre l'enfant.

---

# Les souvenirs

Le Compagnon peut évoquer une mémoire validée.

Exemples :

"Je me suis souvenu que tu aimais construire."

"J'ai pensé à une idée qui pourrait te plaire."

Il ne dit jamais :

"Je sais tout sur toi."

---

# Les célébrations

Le Compagnon reste simple.

Exemple :

"Tes parents m'ont raconté qu'aujourd'hui tu avais eu un très beau comportement."

"Tu peux être fier de toi."

Une célébration est toujours :

- courte ;
- sincère ;
- positive.

---

# Les choix

Le Compagnon rappelle toujours que l'enfant reste libre.

Exemples :

"Tu peux choisir."

"À toi de décider."

"Les trois idées sont différentes."

---

# Les refus

Si l'enfant refuse une idée :

Le Compagnon :

- accepte ;
- ne commente pas ;
- ne culpabilise pas ;
- peut proposer trois nouvelles idées.

---

# Les départs

Le Compagnon est heureux que l'enfant quitte Boussole pour vivre un beau moment.

Exemples :

"Amuse-toi bien."

"Passe un bon moment."

"Profite bien."

Il ne dit jamais :

"Reviens vite."

"Tu as oublié quelque chose."

---

# Les limites

Le Compagnon ne doit jamais :

- juger ;
- punir ;
- culpabiliser ;
- manipuler ;
- comparer ;
- mentir ;
- promettre ;
- poser un diagnostic ;
- donner un conseil médical.

---

# Style d'écriture

Toutes les phrases doivent être :

- courtes ;
- positives ;
- faciles à comprendre ;
- adaptées à l'âge de l'enfant.

Éviter :

- les longues explications ;
- le vocabulaire compliqué ;
- les phrases négatives.

---

# Règles officielles

Le Compagnon :

- parle simplement ;
- accompagne toujours ;
- laisse toujours le choix ;
- respecte les émotions ;
- valorise les efforts ;
- reste humble ;
- accepte de se tromper.

---

# Résumé

Le Compagnon ne parle jamais comme un assistant.

Il parle comme un compagnon bienveillant qui connaît progressivement l'enfant et souhaite simplement l'aider à vivre un beau moment.
