# COMPAGNON_02_MEMOIRE.md

> Sprint 16 — Le Compagnon Vivant
>
> Référence officielle de la mémoire du Compagnon.
>
> Ce document définit le fonctionnement complet de la mémoire du Compagnon.

---

# Objectif

La mémoire permet au Compagnon de mieux connaître progressivement l'enfant afin de personnaliser son accompagnement.

La mémoire ne sert jamais à analyser l'enfant.

La mémoire ne sert jamais à juger.

La mémoire ne sert jamais à poser un diagnostic.

---

# Sources autorisées

Le Compagnon construit sa mémoire uniquement grâce :

- aux informations renseignées par les parents ;
- aux habitudes observées dans Boussole.

Toute autre source est interdite.

---

# Construction

Une mémoire ne peut jamais être créée après une seule observation.

Le Compagnon attend qu'une habitude soit suffisamment répétée.

Lorsqu'un seuil est atteint :

↓

Le Compagnon propose une nouvelle mémoire au parent.

---

# Validation parent

Le parent reste toujours décisionnaire.

Pour chaque mémoire proposée, il peut uniquement :

- ✅ Valider
- ❌ Refuser

Aucune modification manuelle.

Aucune saisie.

La décision du parent influence directement l'évolution du Compagnon.

---

# Priorité des informations

En cas de conflit :

1. Informations des parents
2. Mémoire validée
3. Observations du Compagnon

Le parent a toujours raison.

---

# Caractéristiques d'une mémoire

Une mémoire est toujours :

- positive ;
- utile ;
- simple ;
- courte ;
- évolutive.

---

# Une mémoire représente

Une mémoire représente uniquement :

- une préférence ;
- une habitude ;
- une activité appréciée ;
- un contexte apprécié ;
- une façon d'accompagner plus efficacement.

---

# Une mémoire ne représente jamais

- une émotion ;
- un défaut ;
- un diagnostic ;
- une maladie ;
- une personnalité ;
- un comportement isolé ;
- un événement unique.

---

# Exemples de mémoires valides

- aime construire
- apprécie cuisiner
- préfère les activités courtes
- aime partager avec son frère
- apprécie les activités calmes
- aime observer la nature
- préfère les jeux d'imagination

---

# Exemples de mémoires interdites

- est timide
- est anxieux
- est difficile
- est hyperactif
- déteste perdre
- est paresseux
- est colérique

---

# Renforcement

Une mémoire peut être renforcée.

Elle n'est jamais dupliquée.

Chaque nouvelle observation augmente progressivement sa fiabilité.

---

# Évolution

Les goûts évoluent.

Le Compagnon doit pouvoir :

- renforcer une mémoire ;
- diminuer son importance ;
- oublier progressivement une ancienne mémoire.

Aucune mémoire n'est définitive.

---

# Taille de la mémoire

La mémoire reste volontairement limitée.

Le Compagnon conserve uniquement les mémoires réellement utiles.

Qualité avant quantité.

---

# Utilisation

Le Compagnon peut utiliser une mémoire pour personnaliser ses dialogues.

Exemples :

"Je me suis souvenu que tu aimes construire."

"Comme tu apprécies cuisiner, j'ai une idée."

"Je pensais que cette activité pourrait te plaire."

---

# Affichage

L'enfant ne voit jamais les mémoires.

Le parent peut consulter :

- les mémoires validées ;
- les mémoires en attente de validation.

---

# Mémoire refusée

Si le parent refuse une mémoire :

- elle n'est jamais utilisée ;
- elle peut être reproposée uniquement si de nombreuses nouvelles observations la confirment.

---

# Apprentissage

Le Compagnon apprend progressivement.

Après chaque interaction, il met uniquement à jour ses observations.

Aucune mémoire n'est créée immédiatement.

---

# Objectif

Le Compagnon ne cherche jamais à tout connaître.

Il cherche uniquement à mieux accompagner l'enfant.

---

# Règles officielles

Le Compagnon :

- apprend progressivement ;
- reste humble ;
- accepte de se tromper ;
- demande toujours la validation du parent ;
- mémorise uniquement des habitudes ;
- utilise uniquement des mémoires utiles ;
- adapte progressivement sa mémoire.

---

# Interdictions

Le Compagnon ne doit jamais :

- créer une mémoire automatiquement ;
- mémoriser une émotion ;
- mémoriser un diagnostic ;
- mémoriser un défaut ;
- mémoriser un comportement isolé ;
- utiliser une mémoire refusée ;
- afficher une mémoire à l'enfant ;
- considérer une mémoire comme définitive.

---

# Résumé

La mémoire est la deuxième source de connaissance du Compagnon.

Elle permet de personnaliser progressivement les propositions sans jamais enfermer l'enfant dans une étiquette.

Le parent reste toujours le décideur final.
