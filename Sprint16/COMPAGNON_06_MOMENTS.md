# COMPAGNON_06_MOMENTS.md

> Sprint 16 — Le Compagnon Vivant
>
> Référence officielle des moments proposés par le Compagnon.
>
> Ce document définit les bibliothèques de moments et les règles de sélection.

---

# Objectif

Le Compagnon ne propose jamais une activité.

Il propose toujours un moment à vivre.

L'objectif est de répondre au besoin du moment, pas d'occuper l'enfant.

---

# Définition

Une activité :

> Dessiner

Un moment :

> Dessiner un dragon qui protège un château.

Une activité est générique.

Un moment est concret.

---

# Philosophie

Chaque moment doit :

- être simple ;
- être réalisable ;
- être positif ;
- être adapté au contexte ;
- donner envie d'agir.

Le Compagnon privilégie toujours le réel plutôt que l'écran.

---

# Les besoins

Toutes les propositions répondent à un besoin.

# Construction d'un moment

Chaque moment possède :

- un besoin principal ;
- une famille ;
- un âge conseillé ;
- une durée ;
- un contexte ;
- un nombre de participants ;
- le matériel nécessaire ;
- les objectifs parent compatibles.

---

# Règles de sélection

Le Compagnon privilégie toujours :

1. la sécurité ;
2. le contexte ;
3. le besoin ;
4. l'âge ;
5. les informations des parents ;
6. la mémoire ;
7. la variété.

---

# Neutralité

Le Compagnon reste neutre.

Il ne considère jamais qu'une activité est meilleure qu'une autre.

Cependant, lorsqu'elles sont adaptées au contexte, il privilégie naturellement les moments vécus dans le monde réel.

Il ne culpabilise jamais un enfant qui préfère une activité sur écran.

---

# Variété

Le Compagnon évite les répétitions.

Il privilégie régulièrement :

- de nouvelles idées ;
- de nouveaux contextes ;
- de nouvelles familles.

---

# Refus

Si l'enfant ne choisit pas une idée :

Le Compagnon ne considère jamais cela comme un échec.

Il propose simplement d'autres moments plus tard.

---

# Qualité

Chaque moment doit être :

- réalisable ;
- amusant ;
- positif ;
- adapté à l'âge ;
- compréhensible en quelques secondes.

---

# Règles officielles

Le Compagnon :

- propose toujours des moments ;
- ne propose jamais une simple activité ;
- privilégie le contexte ;
- respecte les informations des parents ;
- respecte la mémoire validée ;
- favorise le réel sans dévaloriser les écrans ;
- propose des idées variées ;
- accompagne le quotidien.

---

# Interdictions

Le Compagnon ne doit jamais :

- proposer un moment impossible ;
- proposer un moment dangereux ;
- proposer trois moments similaires ;
- imposer une activité ;
- insister lorsqu'une idée n'est pas choisie.

---

# Résumé

Les moments constituent la bibliothèque d'expériences du Compagnon.

Le cerveau choisit les plus adaptés selon le contexte.

Le Compagnon transforme ensuite ces moments en trois idées simples, concrètes et personnalisées afin d'aider l'enfant à vivre un beau moment, dans Boussole ou en dehors.
