# COMPAGNON_01_PROFIL.md

> Sprint 16 — Le Compagnon Vivant
>
> Référence officielle du profil enfant.
>
> Ce document définit toutes les informations que le Compagnon est autorisé à connaître grâce aux parents.

---

# Objectif

Le profil permet au Compagnon de personnaliser ses propositions.

Le profil doit être :

- rapide à compléter ;
- simple à comprendre ;
- facile à modifier ;
- utile au fonctionnement du Compagnon.

Temps cible de création :

**2 à 5 minutes**

---

# Principes

Le profil respecte toujours les règles suivantes :

- aucune question inutile ;
- aucune donnée non exploitée ;
- les informations des parents sont prioritaires ;
- le Compagnon ne demande jamais une information qu'il peut déduire ;
- le profil reste évolutif.

---

# Informations obligatoires

## Prénom

Utilisé dans les dialogues.

---

## Date de naissance

Obligatoire.

Permet de calculer automatiquement :

- âge ;
- anniversaire ;
- évolution de l'âge.

L'âge n'est jamais demandé directement.

- agenda familial ;
- événements familiaux.

## Exemples

- activités salissantes

---

# Priorité des informations

En cas de conflit :

1. Informations renseignées par les parents
2. Mémoire validée du Compagnon
3. Observations du Compagnon

Cet ordre est obligatoire.

---

# Règles de développement

Toute nouvelle information ajoutée au profil doit respecter les conditions suivantes :

- améliore au moins une proposition du Compagnon ;
- améliore au moins un dialogue ;
- améliore l'accompagnement d'un contexte ;
- améliore la personnalisation.

Si aucune condition n'est remplie :

**La question ne doit pas être ajoutée.**

---

# Informations interdites

Ne jamais demander :

- personnalité ;
- qualités ;
- défauts ;
- caractère ;
- diagnostic ;
- texte libre ;
- questionnaire à remplir par l'enfant.

Le Compagnon apprend progressivement grâce à son utilisation.

---

# Résumé

Le profil est la première source de connaissance du Compagnon.

Il doit rester :

- simple ;
- rapide ;
- évolutif ;
- utile ;
- exclusivement orienté vers un meilleur accompagnement de l'enfant.
