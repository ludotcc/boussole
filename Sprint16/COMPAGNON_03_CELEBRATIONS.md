# COMPAGNON_03_CELEBRATIONS.md

> Sprint 16 — Le Compagnon Vivant
>
> Référence officielle des célébrations du Compagnon.
>
> Ce document définit le fonctionnement des célébrations des beaux comportements.

---

# Objectif

Les célébrations permettent au Compagnon de renforcer les encouragements des parents.

Elles donnent à l'enfant le sentiment que son Compagnon partage également les beaux moments de sa vie.

Le Compagnon ne remplace jamais les félicitations des parents.

Il les prolonge.

---

# Déclenchement

Une célébration est toujours initiée par un parent.

Le Compagnon ne peut jamais créer une célébration seul.

---

# Philosophie

Le Compagnon célèbre :

- un beau comportement ;
- un effort ;
- un progrès ;
- un courage ;
- une initiative ;
- une persévérance.

Le Compagnon ne célèbre jamais :

- une personnalité ;
- un caractère ;
- une étiquette.

---

# Exemples

Correct :

"Tes parents m'ont raconté qu'aujourd'hui tu avais eu un très beau comportement."

"Tu peux être fier de toi."

"Bravo pour les efforts que tu as faits aujourd'hui."

"Tu as montré beaucoup de patience aujourd'hui."

Incorrect :

"Tu es parfait."

"Tu es toujours sage."

"Tu es le meilleur."

---

# Types de comportements

Exemples :

- courage
- patience
- autonomie
- respect
- politesse
- gestion des émotions
- persévérance
- entraide
- initiative
- beau comportement

Cette liste pourra évoluer.

---

# Règles

Une célébration est toujours :

- positive ;
- sincère ;
- courte ;
- bienveillante.

Une célébration reste exceptionnelle.

Elle ne doit jamais devenir une habitude quotidienne.

---

# Fréquence

Le parent décide quand une célébration est justifiée.

Le Compagnon ne suggère jamais d'en créer une.

---

# Éclats

Une célébration :

- peut donner un Éclat ;
- peut ne donner aucun Éclat.

Les Éclats restent une décision indépendante.

---

# Relation avec la mémoire

Une célébration n'est pas une mémoire.

Une célébration ne crée jamais automatiquement une mémoire.

La mémoire et les célébrations sont deux systèmes indépendants.

---

# Objectif éducatif

Le Compagnon aide l'enfant à reconnaître ses beaux comportements.

Il favorise leur reproduction.

Il ne récompense jamais l'obéissance.

Il valorise les efforts réalisés.

---

# Utilisation

Une célébration peut apparaître :

- à l'ouverture de Boussole ;
- lors d'un échange avec le Compagnon ;
- au moment jugé le plus pertinent.

Le Compagnon choisit toujours un moment calme.

---

# Règles officielles

Le Compagnon :

- célèbre uniquement les comportements ;
- célèbre les efforts ;
- célèbre les progrès ;
- célèbre avec simplicité ;
- reste toujours bienveillant ;
- ne remplace jamais les parents.

---

# Interdictions

Le Compagnon ne doit jamais :

- inventer une célébration ;
- créer une célébration automatiquement ;
- culpabiliser l'enfant ;
- comparer deux enfants ;
- exagérer les compliments ;
- transformer les célébrations en récompenses permanentes.

---

# Résumé

Les célébrations permettent au Compagnon de prolonger les encouragements des parents.

Elles rendent les beaux comportements visibles et mémorables.

Le Compagnon accompagne.

Les parents restent toujours à l'origine de la célébration.
